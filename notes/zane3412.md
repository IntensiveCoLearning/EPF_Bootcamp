# 2026-04-09
<!-- DAILY_CHECKIN_2026-04-09_START -->

Besu = 一个“以太坊执行层节点软件”

运行 EVM + 执行交易 + 维护状态 + 对外提供 RPC

Besu = 一个完整的 Execution Layer 实现（客户端）

Besu 是一个 **Ethereum Client（以太坊客户端）**

它包含：

-   执行环境（EVM）
    
-   状态存储
    
-   P2P 网络
    
-   API 接口
    

Execution Environment 执行环境

核心就是：

-   EVM（以太坊虚拟机）
    

作用：

-   执行智能合约
    
-   运行 opcode
    
-   计算结果
    

写的solidity都是在这里执行

Storage 状态存储

存链上数据：

-   账户余额
    
-   合约 storage
    
-   nonce
    

P2P Network 节点网络

Besu 不是单机程序，它会：

-   连接其他节点
    
-   同步区块
    
-   广播交易
    

所有节点：

都维护同一份状态（去中心化）

API 接口层

对外提供：

-   JSON-RPC
    
-   WebSocket
    

支持：

-   eth\_call
    
-   eth\_sendTransaction
    
-   debug
    

# Besu vs 其他 EL Client（你必须知道）

| Client | 语言 | 特点 |
| Geth | Go | 最主流 |
| Erigon | Go | 高性能 |
| Nethermind | .NET | 企业 |
| Besu | Java | 企业 + 私链 |
